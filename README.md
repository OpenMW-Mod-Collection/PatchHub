<details>
<summary>Web of Mephala - Entrance Patch (OpenMW)</summary>

Forces you to teleport in the correct location when entering the new fancy Morag Tong HQ

Brute force solution to load order issue or a mod conflict. I'm not sure myself why exactly it happens, but I figured it would be easier to patch it with quick and dirty lua script than debug my mod conflicts even considering everything should just work.

Requires: [Web of Mephala Morag Tong Headquarters](https://www.nexusmods.com/morrowind/mods/55163)  
Install like any other mod out there. Load order doesn't matter.

</details>

<summary>Zerkish Hotkeys Improved - Improvement Patch (OpenMW)</summary>

Fixes:

- Partially fixed log spam introduced in OpenMW 0.51 (it's not 100% clean, but it's uninvasive enough for me to be satisfied)
- Fixed mod crashing if an item registered in the hotbar no longer exist in the game (mod adding it was removed mid-playthrough)
- Hotkey UI no longer shows during dialogue, chargen and loading
- Hotkey UI no longer can be used during werewolf transformation
- Disabled all logging due to how excessive it was

Requires: [Zerkish Hotkeys Improved](https://www.nexusmods.com/morrowind/mods/56583) version b1.6.4  
Override files in the mod's folder with mine.
